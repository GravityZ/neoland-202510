# MyPet

## Introduction

An for pet owners register their pets and their medicar information, to have control on their health history.

## Funcional description

### Use cases

User
- register
- login
- update credentials (username, password)
- update profile (name, email, phone, city, role, licensing number)

Owner (User)
- add pet
- remove pet
- modify pet
- list pets
- add comment log for pet
- remove comment log for pet
- modify commentn for pet
- list logs for pet
- filter veterinaries
- assingn veterinary
- unassing veterinary for a pet

Veterinary (User)

- list assigned pets
- add action log for pet
- remove action log for pet
- modify action log for pet
- list logs for pet
